# ant-vue

> A Vue.js project


## 相关技术

vue@2.5.2 + vuex@3.0.1 + vue-router@3.0.1 + axios@0.18.0 + websocket + ant-design-vue@1.1.1 + echarts@4.2.0


### 如何运行

npm install

npm start

访问 localhost:3000
